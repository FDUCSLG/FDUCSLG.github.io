# 讲座01：Rust沙龙

![Rust](/Rust.png)

## 活动介绍

北京大学学生 Linux 俱乐部 2022 秋第 4 次活动 ：Rust沙龙（FDUCSLG分会场）

主要聚焦于Rust语言介绍、特性和学习路径，并介绍Rust在跨平台方面的应用。

## 分享嘉宾

**Mike Tang**  
Rust 中文社区创始人之一。  
介绍 Rust 的简介，历史和特性。

**谷雨**  
北大学生，VS Code 配置助手(v4.vscch.tk )开发者。  
介绍 Rust 开发的心得（以 vscch 为例），以及对 C++ 近年来发展的看法。
