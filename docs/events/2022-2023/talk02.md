## Xmake

构建系统的后起之秀，广受好评。

> Xmake = Build backend + Project Generator + Package Manager + [Remote Distributed] Build + Cache

## 墨干编辑器

GNU TeXmacs 的衍生版，科技编辑平台，所见即所得，并可以高质量导出至 LaTeX。

> 墨干编辑器 = LaTeX + Beamer + Emacs + JupyterLab + 几何画板
